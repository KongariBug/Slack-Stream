# Slack Stream
## Introduction
Slack Stream is a slack client to show the all messages in one window.


## How to use
1. isntall Node.js
2. git clone this repository and run following commands

```shell
cd path-to-this-reposritory
npm install

# set your api token
vi src/ts/token.ts

gulp build
gulp cp-lib
electron .
```
